Battleship

The classic naval combat game.

Project Summary

Our project recreated the classic naval combat game, Battleship, using two FPGA boards. Each board has its own custom RISC-V processor using the RV32-I ISA, extended with our own custom instructions. They also have their own Picture Processing Unit (PPU), UART module, PS/2 keybaord controller, and custom accelerator. With this we run our own C implementation of battleship on each board at the same time, where the baord communicate their moves to each other though UART. When its not your turn, there is a Monte Carlo algorihtm running on your board using the accelerator to calculate your next best move incase you need the extra help. All of this together allows for a completely custom two player Battleship experience, completely implemented on FPGA's.
